# Project Name

## Description
이 프로젝트는 Python을 사용하여 데이터를 크롤링하고, 크롤링한 데이터를 웹페이지에서 시각화

## Structure
- `crawler/`: 크롤링 관련 코드와 데이터 처리
- `web/`: HTML, CSS, JavaScript 파일로 구성된 웹 인터페이스
- `venv/`: Python 가상환경
- `requirements.txt`: 프로젝트에서 필요한 패키지 목록
